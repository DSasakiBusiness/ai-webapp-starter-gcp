| `.claude/` | Claude Code エージェント・スキル定義 |

## データフロー

### チャット（LLM 呼び出し）
1. ユーザーがフロントエンドでメッセージを入力
2. Next.js → NestJS API (`POST /api/llm/complete`)
3. NestJS が OpenAI API を呼び出し
4. レスポンスを DB に保存（会話履歴）
5. フロントエンドに結果を返却

### RAG（検索拡張生成）
1. ドキュメントをアップロード → チャンク分割 → エンベディング生成
2. DocumentChunk テーブルに保存
3. クエリ時にベクトル検索でチャンクを取得
4. 関連チャンクをコンテキストとして LLM に送信

## セキュリティ方針

| レイヤー | 対策 |
|---------|------|
| 入力 | class-validator によるバリデーション |
| 認証 | JWT ベースの認証 |
| 認可 | ロールベースアクセス制御 (RBAC) |
| 通信 | CORS 制限、HTTPS (本番) |
| 秘密情報 | Secret Manager、環境変数 |
| LLM | プロンプトインジェクション対策、出力サニタイズ |
| コンテナ | 非 root ユーザーで実行 |

## 非機能要件目標

| 項目 | 目標 |
|------|------|
| 応答時間（API） | p95 < 500ms（LLM 呼び出し除く） |
| 可用性 | 99.5% (Cloud Run SLA) |
| スケーリング | 0-10 インスタンス（Cloud Run 自動スケール） |
| コスト | 月額 $20-50 (スターター構成) |
